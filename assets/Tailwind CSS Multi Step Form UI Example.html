Tailwind CSS Multi Step Form UI Example

✍ Tailwind CSS
⚇ by larainfo
⌚ 29-11-2021

In this section we will create multi step form ui with tailwind css, tailwind multi step form wizard, multi step form with progress bar, multi step registration form with image example with Tailwind CSS.

Tailwind CSS simple contact us multi step form.

<!DOCTYPE html>

  <head>

        "
      >
        Get in touch
      </h2>

      <p class="max-w-screen-md mx-auto text-center text-gray-500 md:text-lg">

              justify-center

              py-3

              sm:px-6 sm:w-auto sm:justify-start
              title-font
            "

            STEP 1

              inline-flex

              justify-center
              w-1/2
              py-3
              font-medium
              leading-none
              tracking-wider

              sm:px-6 sm:w-auto sm:justify-start
              title-font
              hover:text-gray-900
            "
          >
            STEP 2
          </a>
          <a
            class="
              inline-flex
              items-center
              justify-center
              w-1/2
              py-3

              tracking-wider

              sm:px-6 sm:w-auto sm:justify-start
              title-font
              hover:text-gray-900
            "
          >
            STEP 3
          </a>
        </div>
        <div class="flex flex-col w-full text-center">

                      w-full
                      px-3
                      py-2
                      text-gray-800
                      border

                      outline-none

                      ring-indigo-300
                    "

                    for="phone"
                    class="inline-flex mb-2 text-sm text-gray-800"

                  <input
                    name="phone"

                      w-full

                      py-2

                      border

                      focus:ring
                      ring-indigo-300

                    "
                  />
                </div>

                <div class="flex items-center justify-between">
                  <button
                    class="
                      inline-flex
                      items-center
                      px-6
                      py-2
                      text-sm text-gray-800
                      rounded-lg
                      shadow
                      outline-none
                      gap-x-1
                      hover:bg-gray-100
                    "
                  >
                    <svg
                      xmlns="http://www.w3.org/2000/svg"
                      class="w-4 h-4"
                      fill="none"
                      viewBox="0 0 24 24"
                      stroke="currentColor"
                    >
                      <path
                        stroke-linecap="round"
                        stroke-linejoin="round"

                      px-6

                      ring-indigo-300
                    "
                  >
                    Next
                  </button>
                </div>
              </form>
              <!-- form - end -->
            </div>
          </div>
        </div>
      </div>
    </div>
  </body>
